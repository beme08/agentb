agentB

v0.2 A benchmark for evaluating open web agents on realistic knowledge-worker tasks.

"How well can open models actually complete useful browser tasks?" — not "Can I invent AGI?"

What it is

  • 60 tasks across search, shopping, research, productivity, and developer workflows.
  • A pluggable Agent interface — shipped with stub and openrouter agents; bring your own by subclassing agents.base.Agent.
  • Scoring: 0.4 * step_coverage + 0.4 * critical_actions + 0.2 * final_answer, plus a built-in failure-type classifier.
  • Reference-answer evaluation with per-task page snapshots for reproducibility.

Quick start

git clone https://github.com/<you>/agentb
cd agentb
pip install -e .[dev,validate]
python3 run_benchmark.py --agent stub --all
python3 leaderboard/generate.py traces/results.json

Failure taxonomy

LabelTrigger
constraint_failureMissed a hard requirement (low critical_actions).
retrieval_failureGot to the right area, didn't find the specific data point.
planning_failureWrong action sequence (low steps).
hallucinationRight steps, wrong final answer.
execution_failurePartial coverage on multiple components.
