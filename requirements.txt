# Core
# --------
# agentB itself has no runtime dependencies. The two shipped agents
# (stub and openrouter) only need:
#
#   - Python 3.10+ standard library
#   - For OpenRouterAgent: openai>=1.0  →  pip install agentb[openai]
#
# Optional extras:
#
#   pip install agentb[validate]   # for scripts/validate_tasks.py
#   pip install agentb[dev]       # for pytest + ruff
#
# Heavy browser-using agents (Browser-Use, OpenAI CUA) are out of scope
# for this release; see docs/PLANNED_AGENTS.md.
